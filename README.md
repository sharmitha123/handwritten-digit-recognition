This project demonstrates handwritten digit recognition using the MNIST dataset.
The script automatically downloads MNIST using Keras, processes the images, and runs a simple CNN-style forward pass to show how digit prediction works.

How to run

Install requirements:

pip install tensorflow matplotlib numpy scipy


Run the script:

python mnist_keras_infer.py
Downloading data from https://storage.googleapis.com/tensorflow/tf-keras-datasets/mnist.npz


This will display a digit and print the predicted and actual label.
